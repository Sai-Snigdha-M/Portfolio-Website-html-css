## Cloning a Website Tricks

- To know what color, background-color, font-type, font-size is used in a website:  
  - Inspect > Responsiveness tab > Zoom the dimension to your comfort > Click on the tag container whose background color you want to know.
  
- Profile photo tips:  
  - Use Google Nano to make your sample pic look like a professional 1:1 aspect ratio.
  
- Symbols:  
  - Use SVG icons ([https://www.svgrepo.com/](https://www.svgrepo.com/)), JPEG, or PNG images.  
  - Best Website: Simple Icons ([https://simpleicons.org/](https://simpleicons.org/)).
  
- Useful tool:  
  - CSS Color Filter Generator website (very useful for the filter property in CSS).
